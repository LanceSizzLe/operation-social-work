## ğŸ“‹ Overview

**Operation: Social Work** is a browser-based educational game designed to help final-year Social Work students develop and practice essential practicum competencies in a safe, engaging environment. Students play as **Agent Casey Casefile**, a trainee operative navigating the Agency Operations HQ while completing missions that mirror real-world Social Work tasks.

### Purpose
- Strengthen professional readiness for practicum placement
- Provide low-stakes practice environment for critical skills
- Reduce practicum anxiety through gamified rehearsal
- Support reflective learning aligned with BSWP471 outcomes
